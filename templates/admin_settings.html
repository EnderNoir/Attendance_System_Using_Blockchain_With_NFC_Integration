Email Notifications
Send attendance receipts to students and session summaries to teachers via Gmail SMTP.
How to Set Up Gmail — Step by Step
1
Go to your Google Account
Open a browser and go to myaccount.google.com using the Gmail account you want to use for sending emails.
2
Turn on 2-Step Verification
Click Security on the left side → find 2-Step Verification → click it and follow the steps to turn it ON.
⚠ This step is required — App Passwords will not appear without it
3
Create an App Password
While still on the Security page, use the search bar at the top and search for "App Passwords" → click it → sign in again if asked.
Under Select app choose Mail, under Select device choose Windows Computer → click Generate.
4
Copy the 16-character password
Google will show a password like abcd efgh ijkl mnop. Copy it immediately — you will not be able to see it again. Paste it into the App Password field below.
⚠ Do NOT use your regular Gmail password — it will not work
5
Fill in the fields below and save
Set these values in the form below:
SMTP Host smtp.gmail.com
SMTP Port 587
Gmail Address Your Gmail address e.g. davs.cvsu@gmail.com
App Password The 16-character password from Step 4
6
Test before going live
After saving, scroll down to Send Test Email, enter your own email address and click Send Test. If you see a ✓ green message, everything is working correctly. Check your inbox for the test email.

What Gets Sent
Overview of email notifications sent by DAVS.
Student Receipt
  • Sent on every card tap
  • Includes: name, subject, section, instructor
  • Attendance status (Present / Late / Absent / Excused)
  • Date and time of tap
  • Blockchain TX hash + Block # as tamper-proof proof
Teacher Summary
  • Sent when session ends
  • Session details: subject, section, time slot
  • Present / Late / Absent / Excused counts
  • Full student list with individual status
  • Blockchain TX hashes for all records
 Student receipt emails are only sent if the student has an email address registered in their profile. Teacher summary emails require the teacher's account to have an email address set.
